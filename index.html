Sports Analytics · R · Regression
LA Kings NHL Trade Value Model

Built a multi-variable linear regression model to predict NHL trade value using a historical dataset of ~2,800 traded contracts. Engineered a draft-pick value table mapping all 224 picks to a composite score, then joined it across four relational trade-data tables. Model features include age at trade, goals, assists, average ice time, AAV, and position. Applied the model to the current Kings roster to rank players by predicted trade value.

Key findings: age has a strong negative effect (−14.2 pts/year), while AAV and ice time are significant positive predictors. Quinton Byfield ranked highest on predicted trade value; Trevor Lewis and Kyle Burroughs lowest. R² ~0.097 — expected for a noisy market.

Optimization · Python · ILP
MLB DraftKings Lineup Optimizer

Built a Binary Integer Program in Python using Pyomo and GLPK to generate optimal DraftKings daily fantasy lineups. Constraints enforce salary cap ($50K), positional requirements (P, C, 1B, 2B, 3B, SS, 3×OF), and realistic lineup diversity. Two modes: an oracle "cheat code" lineup using actual game points, and a predictive lineup using historical player averages.

Python · SQL · Tableau · Business Report
HAMA Martial Arts — Membership Revenue Analysis

End-to-end business analysis for High Altitude Martial Arts, built with Python (pandas, matplotlib), SQL, and Excel, then visualized in a Tableau dashboard. Analyzed membership revenue across seven dimensions — quarterly/monthly trends, average purchase point, plan revenue ranking, sign-up channel, day-of-week timing, discount rates, and demographics.

Delivered as a client-facing business report with key findings and recommendations: revenue peaks mid-year (Q3 '24 at $49K) and dips in winter; Gold is the top plan ($157K); 66% of sign-ups are digital; and purchases peak Thursday. The report ties each finding to a concrete action on pricing, promos, staffing, and marketing.
